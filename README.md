# Quantum-Fourier-Tranformer
A researchathon project exploring crosslayer optimization in quantum computing combining pulse engineering, thermal constraints, compiler design, and risk-aware evaluation into a unified hardware-aware framework to minimise noise so focused on bridging the gap between theoretical quantum circuits and real-world superconducting hardware constraints.
DCG v4 – Hardware-Aware Quantum Compiler Framework
A hardware-aware quantum compilation framework developed during a Researchathon, focused on bridging the gap between theoretical quantum circuits and real-world superconducting hardware constraints.
Key Features
DRAG Pulse Engineering for leakage reduction
Thermal Budget Modeling (20 mK regime)
Static ISA Compiler with waveform slot optimization
CVaR Worst-Case Risk Analysis
End-to-End Hardware Simulation with visualization
